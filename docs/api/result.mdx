# ScenarioResult API

The `ScenarioResult` class represents the outcome of a scenario test run, providing detailed information about the test results.

## ScenarioResult Class

```python
from scenario import ScenarioResult

# Access result properties
if result.success:
    print("Test passed!")
    print(f"Met criteria: {result.met_criteria}")
else:
    print(f"Test failed: {result.failure_reason}")
    print(f"Triggered failures: {result.triggered_failures}")
```

## Properties

<ResponseField name="success" type="bool">
  Whether the scenario test passed
</ResponseField>

<ResponseField name="conversation" type="List[Dict[str, str]]">
  The full conversation history between the testing agent and the agent under test
</ResponseField>

<ResponseField name="artifacts" type="Dict[str, Any]">
  Additional artifacts collected during the test
</ResponseField>

<ResponseField name="failure_reason" type="Optional[str]">
  Reason for failure, if the test failed
</ResponseField>

<ResponseField name="met_criteria" type="List[str]">
  List of success criteria that were met
</ResponseField>

<ResponseField name="unmet_criteria" type="List[str]">
  List of success criteria that were not met
</ResponseField>

<ResponseField name="triggered_failures" type="List[str]">
  List of failure criteria that were triggered
</ResponseField>
